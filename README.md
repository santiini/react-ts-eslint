# React-TypeScript

## React-Typescript-Eslint 使用规范

### commitlint 规则

`git commit` 的标准格式如下：

```bash

<type>[optional scope]: <description>

[optional body]

[optional footer]

## optional body 和 optional footer 通常我们都不会用到
## 举例
git commit -m 'docs: update README.md about typescript'

```
